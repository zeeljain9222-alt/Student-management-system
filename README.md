# Student Management System

## Description
A basic Student Management System developed using Java. This project allows users to enter details of multiple students, store them using ArrayList, and display student records.

## Technologies Used
- Java
- Object-Oriented Programming (OOP)
- ArrayList
- DataInputStream

## Features
- Enter details of multiple students
- Store student objects using ArrayList
- Display all student records

## Concepts Covered
- Classes and Objects
- User Input Handling
- Loops
- Collections Framework
- Exception Handling
